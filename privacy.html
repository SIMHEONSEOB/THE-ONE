개인정보처리방침

최종 업데이트: 2024년 2월 22일

1. 개인정보 수집 및 이용 목적

The One(이하 "본 서비스")은 사용자의 개인정보를 수집하지 않습니다. 본 서비스는 다음과 같은 기능만을 제공합니다:

  • 주식 정보 시각화 및 분석
  • 로컬 스토리지를 통한 데이터 저장 (사용자 기기 내부)
  • 익명의 사용자 경험 향상

2. 수집하는 정보

본 서비스는 다음 정보만을 수집하며, 모두 익명으로 처리됩니다:

  • 쿠키 및 로컬 스토리지: 사용자 환경설정, 추천 종목 기록
  • 사용 로그: 방문 시간, 페이지 조회 기록 (IP 주소 수집 안함)
  • 기술 정보: 브라우저 종류, 화면 해상도 (성능 최적화 목적)

3. 개인정보의 보유 및 이용 기간

수집된 정보는 다음 기간 동안 보유됩니다:

  • 로컬 스토리지 데이터: 사용자가 직접 삭제할 때까지
  • 서버 로그: 30일 후 자동 삭제
  • 분석 데이터: 6개월 후 자동 삭제

4. 개인정보의 제3자 제공

본 서비스는 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다:

  • 법원, 검찰 등 수사기관의 요청이 있는 경우
  • 법령의 규정에 따른 경우
  • 서비스 제공을 위한 필수적인 경우 (사용자 동의 하에)

5. 개인정보의 파기 절차

사용자는 언제든지 자신의 개인정보를 파기 요청할 수 있습니다:

  • 로컬 스토리지: 브라우저 설정에서 직접 삭제
  • 서버 데이터: 문의하기 페이지를 통해 요청
  • 파기 확인: 7일 이내 이메일로 통지

6. 쿠키 정책

본 서비스는 다음 목적으로 쿠키를 사용합니다:

  • 사용자 환경설정 저장
  • 서비스 성능 최적화
  • 사용자 경험 개선

사용자는 브라우저 설정을 통해 쿠키를 거부할 수 있습니다.

7. 개인정보보호 책임자

담당자: 개인정보보호 책임자

이메일: (추후 공개 예정)

연락처: GitHub Issues를 통한 문의

8. 정책 변경

본 개인정보처리방침은 다음과 같은 경우 변경될 수 있습니다:

  • 법령 및 정책의 변경
  • 서비스 기능의 추가/변경
  • 회사 정책의 변경

정책 변경 시 최소 7일 전 웹사이트를 통해 공지합니다.

9. 권리와 의무

사용자는 다음 권리를 가집니다:

  • 개인정보 열람 요구권
  • 개인정보 정정 요구권
  • 개인정보 삭제 요구권
  • 처리 정지 요구권
